Questo cartella include script e modelli Simulink, utili per lo sviluppo e l’analisi del progetto.
Di seguito, una descrizione dei file e del loro utilizzo:
-matlab_progetto.m
    Script MATLAB che esegue il modello matematico principale del progetto. Eseguire questo file per ottenere le figure
    riportate nella relazione. Lo script utilizzerà automaticamente il file simulink_puntiOpzionali.slx per completare 
    le simulazioni necessarie ai punti opzionali 2 e 3. Inoltre l'esecuzione di questo script aprirà automaticamente 
    gli altri due file Simulink. 
-simulink_progetto.slx
    Modello Simulink principale. Eseguire per simulare il sistema non linearizzato e ottenere la figura 13 della relazione.
